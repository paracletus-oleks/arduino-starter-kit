Das zweite Projekt aus dem Arduino Starter Kit trägt den Namen „Raumschiff-Steuerzentrale“.
Kurz gesagt, die Aufgabe ist folgende:

- Im geschlossenen Stromkreis, aber ohne Betätigung des Druckknopfes, leuchtet die gelbe LED auf unserem „Panel“;

- beim Drücken des Knopfes sollen zwei rote LEDs abwechselnd aufleuchten und erlöschen.
